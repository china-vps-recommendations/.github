## 先搞清楚：移动用户选 VPS，线路是第一优先级

选 VPS 这件事，移动用户其实比电信、联通用户更难——因为市面上主打 CN2 GIA 的产品，大多数是电信优先的，移动去程往往会走普通路由绕了一大圈。

对移动用户来说，值得关注的线路主要是：

**CMIN2（移动精品网）**：这是移动自家的高等级精品网络，类似电信的 CN2 GIA，走 CMIN2 的 VPS 对移动用户的延迟优化最直接，稳定性也最好。

**CMI**：中国移动国际线路，比 CMIN2 低一个档次，但也算优化线路，延迟一般在合理范围内。

**AS9929（联通精品网）**：虽然是联通的，但部分架构下移动也能走到，一些三网回程走 9929 的产品，移动体验也不差。

**三网 CMI/CN2/CU2 直连**：香港机房常见的配置，电信走 CN2，联通走 9929，移动走 CMIN2，三条路各走各的，效果最均衡。
